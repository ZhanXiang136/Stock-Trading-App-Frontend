# 💹 Trading Bot Dashboard

This React.js dashboard visualizes your trading bot's performance vs. the S&P 500 and NASDAQ over the past 30 days. It consumes a backend API that provides bot equity and index return data.

---

## 📦 Tech Stack

- React (Create React App)
- Chart.js via `react-chartjs-2`
- Fetch API

---

## 🧪 Example Output

Multi-line chart comparing:
- 📈 Your bot's equity
- 📉 S&P 500 cumulative %
- 📉 NASDAQ cumulative %
